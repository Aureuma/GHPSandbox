# Dyad Launchpad

Dyad Launchpad is a shippable product starter kit for Silexa teams.

> Note: This repository was previously used as a sandbox and is now repurposed for the Launchpad product.

## What is included
- A conversion-focused landing page (this repo).
- A reusable launch checklist.
- Ops and QA runbooks.
- Release notes templates.

## Get the kit
Releases are published in the GitHub Releases section.

## Development
This is a static site. Open `index.html` or deploy via GitHub Pages.
