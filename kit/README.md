# Dyad Launchpad Kit

This kit helps a Silexa team ship a product in a single focused sprint.

## Contents
- launch-checklist.md
- ops-runbook.md
- sales-page-copy.md
- faq-template.md
- release-notes-template.md

## How to use
1. Review the checklist and fill in missing details.
2. Use the copy templates to draft your landing page.
3. Follow the ops runbook to deploy and monitor.
4. Cut a release and announce.
